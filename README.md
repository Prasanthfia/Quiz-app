# 🧠 Quiz App

An interactive multiple-choice quiz on web development fundamentals, built with HTML, CSS, and JavaScript.

## Live Demo
Open `index.html` in any browser — no installation needed.

## Features
- 8 multiple choice questions on HTML, CSS, and JavaScript
- Instant right/wrong feedback with color coding
- Progress bar showing quiz completion
- Live score tracker
- Final results screen with emoji feedback
- Restart option

## Built With
- HTML5
- CSS3
- Vanilla JavaScript
